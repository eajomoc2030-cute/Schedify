# Schedify
**Schedify** is a simple, intuitive, and user-friendly weekly organizer designed for students. It helps manage academic responsibilities and personal activities in one place, making it easy to track deadlines, mark tasks as done, and stay organized throughout the week.

## Features
- **Add Tasks Easily:** Quickly add both academic and non-academic tasks.  
- **Task Details:** Set a description, deadline, and category for each task.  
- **Task Overview:** View all tasks in the order they were added.  
- **Task Status:** Mark tasks as **Pending** or **Done**.  
- **Edit or Delete Tasks:** Update task information or remove tasks when needed.  
- **User-Friendly Interface:** Simple and easy to navigate for beginners.  

## How to Run
1. Open the Python file [Schedify](https://onlinegdb.com/ar2xV-jSBD). 
2. Follow the on-screen prompts to add, update, view, or delete tasks.  
3. Track your weekly schedule efficiently and never miss a deadline.  

## Benefits
- Helps manage time effectively  
- Reduces stress by keeping all tasks organized  
- Improves academic performance through better planning  
- Encourages accountability with task completion tracking

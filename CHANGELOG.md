## Version v1.2.0 – December 4, 2025
We removed the extra features and kept only the main **Key Features of Schedify**:
- Add academic and non-academic tasks  
- Set task descriptions, deadlines, and categories  
- View all tasks in the order they were added  
- Mark tasks as Done or Pending  
- Update or delete tasks  

---

## Version v1.1.0 – August 15, 2025
- User can view a countdown showing days remaining until a task's deadline  
- User’s tasks are automatically sorted by deadline  

---

## Version v1.0.1 – August 15, 2025
- User can mark tasks as Done or Pending  
- User sees a motivational message when marking a task as done  
- User can update or delete existing tasks  

---

## Version v1.0.0 – August 15, 2025
- User can add academic and non-academic tasks with descriptions, deadlines, and categories  
- User can view a list of all tasks for the week  
- User's tasks are automatically saved and loaded (autosave feature)
